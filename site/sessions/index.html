    Les sessions en PHP

    1. Créer une page index.php avec un formulaire dedans :

        <!DOCTYPE html>
    <html lang="fr">
    <head>
        <meta charset="UTF-8">
        <meta http-equiv="X-UA-Compatible" content="IE=edge">
        <meta name="viewport" content="width=device-width, initial-scale=1.0">
        <title>Document</title>
    </head>
    <body>
        <form action="validation.php" method="post">
            <fieldset>Formulaire de connexion</fieldset>
            <label name="lbl_nom">Nom : </label>
            <input type="text" name="nom" pattern="[a-zA-Z]{1,}"></input><br>
            <label name="lbl_prenom">Prénom : </label>
            <input type="text" name="prenom" pattern="[a-zA-Z]{1,}"></input><br>
            <label name="lbl_age">Age : </label>
            <input type="number" name="age"></input><br>
            <input type="submit" name="submit" value="Valider">
        </form>
    </body>
    </html>
    2. Démarrer la session avec la function session_start() : 

        <?php
    session_start();

    ?>
    1. Récupérer les valeurs du formulaire avec des variables php :
        <?php
    session_start();

    if(isset($_POST['submit'])){
        $nom = filter_input(INPUT_POST,'nom',FILTER_SANITIZE_SPECIAL_CHARS);
        $prenom = filter_input(INPUT_POST,'prenom',FILTER_SANITIZE_SPECIAL_CHARS);
        $age = filter_var(INPUT_POST,'age',FILTER_SANITIZE_NUMBER_INT)
        $nom = $_SESSION['nom'];
        $prenom = $_SESSION['prenom'];
        $age = $_SESSION['age'];
        }
    ?>
